# Requirement Clarifier

Use this skill when the request is still too fuzzy to implement safely.

## Workflow

1. Clarify core intent, scope, anti-goals, and edge cases.
2. Check alignment with the project's North Star and constraints.
3. Play the request back as:
   - User story
   - Acceptance criteria
   - Constraints and out-of-scope
4. Finalize handoff docs:
   - `spec.md`
   - `contracts.md`
